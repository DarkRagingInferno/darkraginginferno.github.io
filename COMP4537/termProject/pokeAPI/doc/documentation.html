poke-amar-john is a web api that tries to capture a bit of the magic of pokemon we all know and love.
https://pokeapi-amar-john.herokuapp.com/api/v1

In order to use this API you must sign up and create a trainer in order to use this service.

Afterwords, simply just login

Pokemon

Pokemon are creatures of all shapes and sizes who live in the wild or alongside humans

Pokemon Endpoints

Endpoint Description
GET

/pokemon

Gets a Pokemon
PUT

/pokemon/update

Updates a pokemon's properties such as [name, hp, attack, generation, etc...]
DEL

/pokemon/delete

Deletes a pokemon
POST

/pokemon/create

Trainers

Trainer Endpoints

Endpoint Description
GET

/trainer

PUT

/trainer/update

Updates a Trainer's information [first name, last name, etc...]
DEL

/trainer/delete

POST

/trainer/create

POST

Party
